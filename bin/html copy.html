Definition List

                                
<dl>
    <dt> Sample Title </dt>
    <dd>
        Sample text
    </dd>

    <dt> Sample Title </dt>
    <dd>
        Sample text
    </dd>
</dl>

Explicación

<ol>

"ol" significa "ordered list" y se usa para listas de cosas con una jerarquía de importancia.

<ul>

"ul" significa "unordered list" y se usa para listas de cosas con el mismo valor o que el orden no importe.

<li>

"li", por otro lado, significa "list item" y se usa de la misma manera en ambas listas.

<dl>

"dl" significa "definition list" y se usa para listas de cosas con un valor término-definición o tema-detalle.

<dt>

"dt" significa "definition title" y se usa para definir el título del tema a detallar.

<dd>

"dd" significa "definition description" y es usado para detallar un tema relacionado con el título (dt).

Explicación

<table>

Hoy en día es usada para mostrar datos tabulares, por ejemplo, horarios, precios, comparativas, etc...

<thead>

Sirve para definir el encabezado de la tabla.

<tbody>

Define el cuerpo o contenido de la tabla.

<tfoot>

Usado para poner resumenes, totales o información extra.

<tr>

"tr" significa "table row" y como su nombre indica crea una fila en la tabla.

<td>

"td" significa "table data" y como su nombre indica crea una celda con un dato.

<colgroup>

Crea un grupo de columnas para hacer la personalización de tablas grandes más fácil, uso específico.

<col>

Cada etiqueta col representa una columna entera de la tabla.
