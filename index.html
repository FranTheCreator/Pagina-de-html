Explicación

<ol>

"ol" significa "ordered list" y se usa para listas de cosas con una jerarquía de importancia.

<ul>

"ul" significa "unordered list" y se usa para listas de cosas con el mismo valor o que el orden no importe.

<li>

"li", por otro lado, significa "list item" y se usa de la misma manera en ambas listas.

<dl>

"dl" significa "definition list" y se usa para listas de cosas con un valor término-definición o tema-detalle.

<dt>

"dt" significa "definition title" y se usa para definir el título del tema a detallar.

<dd>

"dd" significa "definition description" y es usado para detallar un tema relacionado con el título (dt).

Enlace y Atributos

<a href="/ruta"> sample text </a> <a href="/ruta" target="_blank"> sample text </a> <a href="/ruta" target="_self"> sample text </a> <a href="/ruta" target="_parent"> sample text </a> <a href="/ruta" target="_top"> sample text </a> <a href="/ruta"> sample text </a> <a href="/ruta" rel="noopener nofollow noreferrer"> sample text </a>

Explicación

<a>

Es usada para enlaces con el atributo hipertext reference o "href" que le indica la ruta que debe seguir.

[target]

El atributo target indica que debe suceder al hacer click en el enlace, hay 4 targets: _blank, abre una pestaña nueva, _self (default), abre en la misma pestaña, _top, es como _self pero para enlaces dentro de iframes de YT, y _parent abre el link dentro del iframe.

[rel]

Por último, el atributo rel establece que relación tiene el destino del enlace con tu pagina. Es recomendado poner "noopener" para que tu página no sea modificada al abrir el enlace, "noreferrer" para que a la otra página no se le registre de donde viene el usuario y "nofollow" para que el SEO no distribuya el posicionamiento de tu página con la otra (página del enlace).

Imágenes

<img src="imagen.png" alt="imagen de ejemplo"> <img srcset="img-S.png 360w, img-M.png 1000w" sizes="(max-width: 400px) 300px, (max-width: 1500px) 1000px" alt="imagen de ejemplo"> 
                                
<picture>
    <source  media="(max-width: 400px)" srcset="img-s.jpg" type="image/jpg">
    <source  media="(max-width: 1500px)" srcset="img-m.jpg" type="image/jpg">
    <img src="fallback-img.png" alt="imagen de ejemplo">
</picture>

Explicación

<img> & [src]

La etiqueta <img> es usada para mostrar imágenes desde html y el atributo [src] para indicar dónde está almacenada la imagen.

[alt]

El atributo alt sirve para dar una descripción de la imagen para las personas no videntes o en caso de que no cargue la imagen.

[srcset]

Es una lista de imágenes con su ancho ideal Ej:(360w = 360 píxeles de ancho, etc)..

[sizes]

Le dice al navegador cuánto espacio ocupará la imagen según el tamaño de pantalla Ej:(Si la pantalla es de hasta 400px → la imagen ocupará 300px.)..

<picture> & <source>

Es una etiqueta que te permite cambiar formato, resolución o imagen según condiciones. Va evaluando los <source> desde arriba hacia abajo y si cumplen la condicion del "media" la imagen se muestra, si no, va a la etiqueta img y muestra esa.

    Fallback text

    Fallback text
    <source src="otro-formato de video.aviv">
    <track src="subs-es.vtt" srclang="es" label="Español" default>

Explicación

<video>

La etiqueta <video> es usada para mostrar video desde html y el atributo [src] para indicar dónde está almacenado el video.

<source>

La etiqueta <source> es usada para cambiar de formato de video en caso de incompatibilidad.

<track>

Le otorga al video subtítulos.

[srclang]

Le dice al navegador en que lenguaje están los subtítulos.

[label]

Establece el nombre de los subtítulos, por ejemplo: Español(España).

[default]

Establece al subtítulo como predeterminado al reproducir el video.

<audio>

Se usa para mostrar audio desde html.

[autoplay]

El audio o video se reproducirá automaticamente en caso de tener el atributo muted.

[controls]

Otorga controles de reproduccion y volumen al usuario.

[loop]

Hará que el audio o video vuelvan a empezar una vez finalizados.

[muted]

Establece el volumen del audio o video a cero.

Explicación

<form>

Sirve para solicitar datos al usuario.

<fieldset>

Agrupa elementos relacionados dentro del formulario.

<legend>

AEs una etiqueta que sirve como título del grupo que está dentro del <fieldset>.

<label>

Asocia una descripción con un input.

<input>

Crea un campo que recibe datos.

[type]

Define el tipo de dato que va a recibir el input.

[name]

Es el nombre del campo; ese es el identificador con el que se enviará al servidor.

<select>

Crea una lista desplegable.

<option>

Cada opción que puede elegir el usuario.

[value]

Es el valor que se enviará al servidor si se selecciona esa opción.

<textarea>

Crea un área de texto donde se pueden escribir varias líneas, normalmente usado para comentarios.

<button>

Crea un botón interactivo.
